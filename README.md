## What I'm building right now

### [workout-journal](https://github.com/americanbiscuit/workout-journal)
A personal workout tracker. Logs morning strength + afternoon cardio/metcon, syncs across devices, supports multiple users.
Built with vanilla HTML/JS + Cloudflare Pages Functions + D1 (SQLite). No frameworks.
Live at [workout-journal-85a.pages.dev](https://workout-journal-85a.pages.dev)

### [house-hunt](https://github.com/americanbiscuit/house-hunt)
Private house-hunting portal for me and my partner. Pulls Zillow listings nightly, tracks price changes, renders a friendly Barbie-themed UI.
Built with vanilla HTML/JS + Cloudflare Pages + scheduled GitHub Actions.
